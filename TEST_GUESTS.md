## Примеры запросов в чате

### Поиск по ФИО:
```
Отменить бронирование для Іванов Петро Сергійович
Отменить броню Петренко Марина
Изменить бронирование Сидоров
```

### Поиск по Email:
```
Отменить бронирование ivanov@example.com
Отменить бронирование для kovalenko@example.com
Изменить бронирование melnyk@example.com
```

### Поиск по телефону:
```

```

## Как получить ID бронирования

1. Откройте консоль браузера (F12)
2. Перезагрузите страницу
3. В консоли увидите список гостей с ID:
```
Sample guests for testing:
  • Іванов Петро Сергійович (BKG-ABC123)
  • Петренко Марина Олександрівна (BKG-XYZ789)
  ...
```

4. Скопируйте нужный ID для тестирования

## Статусы бронирований

- **confirmed** - активное бронирование (можно отменять/редактировать)
- **cancelled** - отменено
- **completed** - завершено (гость выехал)

## Типы номеров

Гости размещены в следующих типах номеров:
- Стандартний номер (1500 $ за ночь)
- Покращений номер (2000 $ за ночь)
- Делюкс (2500 $ за ночь)
- Сімейний номер (3000 $ за ночь)
- Люкс (4000 $ за ночь)

## Даты бронирований

Даты генерируются автоматически в диапазоне:
- От: 10 дней назад
- До: 30 дней вперед
- Продолжительность: от 2 до 7 ночей

Таким образом, часть бронирований уже завершена (статус completed), часть активна (confirmed), и некоторые могут быть отменены (cancelled).

## Примечание

Все данные хранятся в localStorage браузера. Чтобы сбросить данные и перегенерировать гостей:
1. Откройте консоль браузера (F12)
2. Выполните: `localStorage.removeItem('hotel_bookings')`
3. Перезагрузите страницу
